🤖 What is Claude?

Claude is an AI assistant built by Anthropic. The Diocese holds a Claude for Work (Team) licence, which means your conversations are never used to train the AI and all data is covered by a formal Data Processing Agreement. These guides explain how each team can use it day-to-day — from drafting letters to managing emails, analysing budgets, and automating routine tasks.
